Debug

In Chameleon Twist, there exists a nulled out flag for a debug mode. At the hex address 0x800f06b0, in normal gameplay, the default value is 0xFFFFFFFF. However, if the value is set to 0x00000000, then the debug menu will be able to be triggered.
To trigger debug mode from here, go into Story Mode and start a level. While holding L, in order, press DPAD UP, DPAD DOWN, DPAD LEFT, and DPAD RIGHT.
The value will now be 0x00000001. Because of this, and since you are still holding L, this will reveal the debug menu.

With the menu now up, your controls will change as well.
The DPAD and 3DSTICK both control the player's x and z, with horizontal controlling x and vertical controlling z. The y is controlled by C UP & C Down respectively.
C LEFT & C RIGHT control the room index, warping you to the entrance/exit. If rooms exist outside of the normal room limits, they are not accessible because the room limits wrap at the end. To access those, manual editing will have to be done.
On the left, there is a GUI. The first word is the internal name of the level. The second number is the Room Id. The next 3 numbers are each position values for the player. The last number is thought to be the distance the camera is from the player.
On the right, there is a list of flags per level. These flags are for level events such as doors closing, or a crown requirement being met. These flags save with your file and can be abused to make... silly situations.

The no-clp movement goes by increments of 100, but using the 3DSTICK, you can move in smaller increments with your player speed.
While in debug mode, rooms are surrounded by a collision box, and field segments are surrounded by a checkpoint box. Level collision still applies, however.
Also while in debug mode, if you press START, you can change the current camera angle. The angles in order are TOP DOWN, SIDE VIEW 1, SIDE VIEW 2, and then back to normal.
In all camera modes, you can use Z to zoom out. The camera loops back at camera distance 20000. In SIDE VIEW 1 and SIDE VIEW 1 only, you can press A to rotate the camera counter clockwise, and B clockwise.
